## Digital Hippo
A Next.js project bootstrapped with "create-next-app" using:

- Typescript & typesafe development
- Authentication
- Admin Dashboards with Payload
- Tailwind styling
- NoSQL database utilization with MongoDB
- Deployment with Railway.
